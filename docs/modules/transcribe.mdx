## <Icon icon="terminal"/> Usage
How to transcribe a message anywhere on Discord with Fyre Bot.

If you're inside of a server, you can reply to the message with `>ts` or `>transcribe`.
Otherwise, below you can learn how to use the Transcribe context command.

### <Icon icon="computer"/> PC Usage
To transcribe a voice message on PC, right click the message,
select "Apps", then "Fyre", and finally, click "Transcribe".

### <Icon icon="smartphone"/> Mobile Usage
To transcribe a voice message on mobile, hold down on the message,
select "Apps", then "Fyre", and finally, tap "Transcribe".

### <Icon icon="timer-reset"/> Cooldowns
- Per-user - `15 seconds`
- Per-server - `30 seconds`

Only one cooldown will apply, depending on where the command is used.
Per-server cooldown always takes priority over per-user.
