## <Icon icon="terminal"/> Usage
How to translate a message anywhere on Discord with Fyre Bot.

If you're inside of a server, you can reply to the message with `>tl` or `>translate`.
Otherwise, below you can learn how to use the Translate context command.

### <Icon icon="computer"/> PC Usage
To translate a text message on PC, right click the message,
select "Apps", then "Fyre", and finally, click "Translate".

### <Icon icon="smartphone"/> Mobile Usage
To translate a text message on mobile, hold down on the message,
select "Apps", then "Fyre", and finally, tap "Translate".

### <Icon icon="timer-reset"/> Cooldowns
- Per-user - `5 seconds`
- Per-server - `5 seconds`

Only one cooldown will apply, depending on where the command is used.
Per-server cooldown always takes priority over per-user.

<Info>
Currently, Fyre only supports translating messages into English.
Eventually, we want to support translating into any language, with per-user and per-server defaults.
</Info>
