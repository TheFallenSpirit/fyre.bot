## <Icon icon="shopping-cart"/> How to Buy
To buy a license for your server, use `/buy` in your server, and tap/click the "Purchase" button.

To buy a license for your user account, use `/buy` anywhere on Discord,
switch the tab to "User License", and tap/click the "Purchase" button.

After purchasing a license, Fyre will DM you to confirm it has been activated.
This DM will also include your Order ID, which can be used for billing support and license transfers.

## <Icon icon="list-check"/> Types of Licenses
Fyre offers 2 different types of licenses. One for servers, and one for users.

Below you can learn what each license offers, and how much they cost.

### <Icon icon="message-circle-more"/> Server Licence | $50
The Fyre server license allows you to use **all** of Fyre's features in one server.
Below is a partial list of modules you can use with a server license.
This is not a full list, and you should always look over Fyre's command list before purchasing.

- [VIP](../modules/vip)
- [AFK](../modules/afk)
- [Jail](../modules/jail)
- [Snipe](../modules/snipe)
- [Forms](../modules/forms)
- [Last.fm](../modules/last-fm)
- [Tickets](../modules/tickets)
- [Activity](../modules/activity)
- [Transcribe](../modules/transcribe)
- [Media Mode](../modules/media)
- [Auto Purge](../modules/purge)
- [Auto Responder](../modules/responses)
- [Auto Threading](../modules/threads)
- [Time Converter](../modules/time)
- [Member Backups](../modules/backup)

### <Icon icon="users"/> User Licence | $8
The Fyre user license allows you to add Fyre to your Discord account, and use certain modules across Discord.
In DMs, Group DMs, and servers without Fyre installed (if external commands are allowed).
